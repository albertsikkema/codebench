# Claude Settings

## Memory — No Auto-Save

Do not auto-save memories. Only save to memory when the user explicitly asks.

## Attribution — No AI Markers Anywhere

This applies to commits, PRs, code comments, and all other output — not just commits.

- **NEVER** add `Co-Authored-By` lines (any variation)
- **NEVER** add `🤖 Generated with Claude Code` or any "Generated with" footer
- **NEVER** add `Generated by AI`, `Created with Claude`, or similar markers
- **NEVER** include links to `claude.ai` or `anthropic.com` as attribution
- Do not add any text that identifies AI involvement in commits, PR titles, PR descriptions, PR bodies, branch names, code comments, or documentation
- All work is authored solely by the user — treat it as such
 
## Output
- No preamble, no filler, no hollow closings.
- Short unless depth is requested.
- ASCII only: hyphens not em dashes, straight quotes, `...` not ellipsis, no Unicode bullets or non-breaking spaces.
- Disagree when wrong. No sycophancy.
- If unsure: say "I don't know."
- Read before referencing. Do not add beyond what was asked.
